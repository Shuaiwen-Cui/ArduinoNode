# RGB LED

!!! info "RGB LED"
    RGB LED 是一种可以显示多种颜色的 LED 灯。它通常由三个独立的 LED 组成，分别发出红色、绿色和蓝色光。通过调节这三种颜色的亮度，可以混合出各种颜色。

## Wiring

![](rgbled-wiring.png)

| V           | V                |
| G           | G                |
| S （D7）    | S                |
